Backbone

This demonstrates how to isolate a jquery plugin behind a Backbone View coupled to a Model in order to gain finer control on the output. The View updates the Model and renders asynchronously once

Note: I used a setTimeout to render the Backbone view for the sake of simplicity (this page is entirely controlled by Angular). History length is limited to 5 entries by default. The History collection exposes methods to the View so that all history-related logic is done internally. Please take a look on how separated the concerns are between model, collection (and its model), and the view.
